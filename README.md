# lark_robot
this is a repository builded by my little-cute and me. these apps are implemented by lark-feishu-api.

## 提交代码
```
1. git add . // 添加所有变动文件
2. git commit -m 'feat: xxx' // 保存到本地
3. git push origin main // 推送到远程main分支上
```

## 代码功能
* `server.py`: 交互式机器人代码
* `xiache.py`: 知乎
* `weather.py`: 获取天气的代码
* `crawer`: 一些爬虫相关代码
